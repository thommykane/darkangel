# Dark Angel Clothing

Luxury fashion e-commerce website built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion**

## Project Structure

```
src/
  app/          # Pages and layout
  components/   # UI components
  lib/          # Utilities and product data
public/
  logo/         # Brand logo
  shirts/       # Product images (auto-loaded)
```

## Deployment

Deploy to [Vercel](https://vercel.com) with zero configuration.
